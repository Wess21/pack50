# 🚀 LLM Providers - Quick Start

## 1️⃣ Откройте новый интерфейс

```
http://localhost:3000/admin.html
```

Логин: `admin` / `changeme`

## 2️⃣ Добавьте провайдер

Нажмите кнопку **"+ Добавить провайдер"**

### Для vsellm.ru (рекомендуется):

```
┌─────────────────────────────────────┐
│ Добавить провайдер                  │
├─────────────────────────────────────┤
│ Название: VSE LLM - Claude          │
│ Тип: OpenAI Compatible              │
│ API Key: ваш-ключ-от-vsellm         │
│ API Base URL:                       │
│   https://api.vsellm.ru/v1          │
│ Модель:                             │
│   anthropic/claude-haiku-4.5        │
│ ☑ Сделать активным провайдером      │
│                                     │
│ [Сохранить]  [Отмена]               │
└─────────────────────────────────────┘
```

## 3️⃣ Протестируйте

Нажмите кнопку **"Тест"** на карточке провайдера:

```
✅ Провайдер работает корректно!
```

или

```
❌ Ошибка: Invalid API key
```

## 4️⃣ Переключайтесь между провайдерами

```
┌──────────────────────────────────────────────────┐
│ VSE LLM - Claude          [АКТИВЕН]              │
│ OPENAI  🌐 api.vsellm.ru  🤖 claude-haiku-4.5    │
│                          [Тест] [Изменить] [×]   │
├──────────────────────────────────────────────────┤
│ OpenAI GPT-4o                                    │
│ OPENAI  🤖 gpt-4o                                │
│     [Активировать] [Тест] [Изменить] [×]         │
├──────────────────────────────────────────────────┤
│ Anthropic Direct                                 │
│ ANTHROPIC                                        │
│     [Активировать] [Тест] [Изменить] [×]         │
└──────────────────────────────────────────────────┘
```

Нажмите **"Активировать"** для переключения. Изменения применяются мгновенно!

## 5️⃣ Готово!

Бот теперь использует выбранного провайдера для всех запросов.

## 🔥 Сценарии использования

### Сценарий 1: Экономия бюджета

1. Добавьте провайдер с `anthropic/claude-haiku-4.5` через vsellm.ru
2. Активируйте его
3. Наслаждайтесь низкими ценами

### Сценарий 2: Максимальное качество

1. Добавьте провайдер с `anthropic/claude-sonnet-4.5` через vsellm.ru
2. Активируйте его
3. Получайте лучшие ответы

### Сценарий 3: Резервирование

1. Добавьте несколько провайдеров (vsellm.ru + OpenAI direct)
2. Если один не работает - переключитесь одним кликом
3. Бизнес продолжает работать

### Сценарий 4: A/B тестирование

1. Добавьте Claude и GPT провайдеров
2. Переключайтесь между ними
3. Сравните качество ответов

## 🆘 Помощь

### Провайдер не проходит тест?

✅ Проверьте API ключ
✅ Проверьте API Base URL (должен заканчиваться на `/v1`)
✅ Проверьте баланс на счете провайдера

### Бот не отвечает?

✅ Убедитесь что хотя бы один провайдер активен (зеленая плашка)
✅ Протестируйте активного провайдера кнопкой "Тест"

### Хочу вернуться к старому интерфейсу

Откройте: http://localhost:3000/index.html
