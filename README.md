# Pack50 - AI-Powered Telegram Bot for Russian SMB Market

Умный AI-ассистент для бизнеса с поддержкой RAG, multi-provider LLM, и автоматической установкой на VPS.

## 🚀 Quick Start (3 Commands)

```bash
# 1. Download and setup
curl -sSL https://raw.githubusercontent.com/your-org/pack50-bot/main/install.sh | bash
cd pack50-bot

# 2. Configure (interactive prompts)
./configure.sh

# 3. Start!
docker compose up -d
```

Готово! Бот работает на `http://localhost:3000`

## 📚 API Endpoints

### Document Upload
```bash
POST /api/documents/upload
Content-Type: multipart/form-data

# Upload PDF
curl -F "file=@document.pdf" http://localhost:3000/api/documents/upload

# Upload URL
curl -X POST http://localhost:3000/api/documents/url \
  -H "Content-Type: application/json" \
  -d '{"url": "https://example.com/page"}'
```

### Admin Panel
```bash
# Login
POST /api/admin/login
{"username": "admin", "password": "your-password"}

# Get config
GET /api/admin/config
Authorization: Bearer <token>

# Update config
PUT /api/admin/config
Authorization: Bearer <token>
{"active_model": "gpt-4o", "anthropic_api_key": "sk-..."}

# Analytics
GET /api/admin/analytics?days=30
Authorization: Bearer <token>
```
