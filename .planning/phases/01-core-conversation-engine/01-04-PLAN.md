    - Add WEBHOOK_SECRET: z.string().min(32).optional() (secret token for validation)
    - Add PORT: z.coerce.number().default(3000)

    **Update src/bot/index.ts:**
    - Import express, webhookCallback from grammy, helmet, express-rate-limit
    - Create Express app before bot initialization
    - Add security middleware: app.use(helmet()), app.use(express.json())
    - Add rate limiting for webhook: rateLimit({ windowMs: 60000, max: 100 }) on /webhook route

    Add webhook validation middleware:
    ```typescript
    app.use("/webhook", (req, res, next) => {
      if (!env.WEBHOOK_SECRET) {
        // Development mode (no webhook secret configured)
        return next();
      }

      const token = req.header("X-Telegram-Bot-Api-Secret-Token");
      if (token !== env.WEBHOOK_SECRET) {
        logger.error("Invalid webhook secret token", { ip: req.ip });
        return res.status(403).send("Forbidden");
      }
      next();
    });
    ```

    Add webhook endpoint: app.post("/webhook", webhookCallback(bot, "express"))

    Update startup logic:
    ```typescript
    if (env.WEBHOOK_URL && env.WEBHOOK_SECRET) {
      // Production: use webhook
      const webhookUrl = `${env.WEBHOOK_URL}/webhook`;
      await bot.api.setWebhook(webhookUrl, {
        secret_token: env.WEBHOOK_SECRET,
        drop_pending_updates: true,
      });
      logger.info(`Webhook set to ${webhookUrl}`);

      // Start Express server
      app.listen(env.PORT, () => {
        logger.info(`Server listening on port ${env.PORT}`);
      });
    } else {
      // Development: use long polling
      logger.info("Starting bot with long polling (development mode)");
      await bot.start();
    }
    ```

    Add graceful shutdown for webhook mode: process.once("SIGINT/SIGTERM", stop server and bot)

    Follow research Pattern 3: Webhook with Secret Token Validation. Secret token prevents fake update injection (BOT-04).
